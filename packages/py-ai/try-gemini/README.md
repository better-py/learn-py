## FAQ:

### google cloud 登录问题

- 安装 gcloud:

```ruby
task install:gcp

```

- 登录 gcloud:

```ruby

# 执行此命令, 会自动打开浏览器, 跳转 Google 登录+授权.
task auth:gcp

```

- 继续执行脚本.
