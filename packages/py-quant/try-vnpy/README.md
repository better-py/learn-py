# vnpy

- 量化交易框架

## Development

### Requirements

- vnpy 包很大, 基于 PyQt(pyside6==6.8.2.1) 实现了 GUI 的策略开发环境.

```ruby
# 安装依赖
brew install ta-lib 

# 安装包依赖
uv add ta-lib numpy
uv add vnpy vnpy_ctastrategy vnpy_ctabacktester vnpy_datamanager vnpy_sqlite vnpy_rqdata   

```
