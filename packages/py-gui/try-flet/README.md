# flet

- Flet = Flutter + Python.
- Python 跨平台 app 开发框架: 支持 iOS, Android, macOS, Windows, Linux.

## Quickstart

### install

```ruby
rye install flet

flet --version 
```

### run

```ruby

flet run main.py 
 
```
