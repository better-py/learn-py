# Django 5.x 实测

- 作为 Django 老用户, 很久没使用.
- 验证一下最新版本的特性.

## Quick start:

### init db:

```ruby
task init
```

### create admin user:

```ruby
task init:user  # input: admin/admin
```

### run http server:

```ruby
task run
```


## Manual:


### new project:

```ruby

django-admin startproject mysite
```

### new app:

```ruby

django-admin startapp myapp
```
