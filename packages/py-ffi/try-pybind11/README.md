# Python FFI + C++ 方案

- python 多语言混编方案

> 双向调用

- python 基于 FFI 方案， 调用其他语言的库
- 其他语言基于 FFI 方案， 调用 python 的库

## Python + C++

- pybind11: 首选方案, 支持 `C++11` 标准
- [pybind11: docs](https://pybind11.readthedocs.io/en/stable/)
